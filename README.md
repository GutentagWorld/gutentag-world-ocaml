# gutentag-world-ocaml

> Gutentag, World! in OCaml

## Run

```
ocaml main.ml
```


## Install

```
opam install gutentag-world
```


### Usage

## Usage

```sh
dune exec gutentag-world
```

Or use in your project:

```ocaml
let () = print_endline (Gutentag_world.gutentag_world ())
```
